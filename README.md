## Python3 Code Helper

- **visualize.html** – paste code, pick Python 3.14, and step through execution (stack/heap/print output). Supports sharing links and embedding.
- **live.html** – collaborative mode to co-view traces in real time (Python/JS options), with synced stepping and chat.

### Run locally
```
docker compose up -d
open http://localhost:8003/v5-unity/visualize.html
```
